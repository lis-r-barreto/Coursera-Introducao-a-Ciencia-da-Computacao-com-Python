**Q2.** Como armazenar em uma nova variável a quantidade total de caracteres das três variáveis abaixo?

```python
cumprimento = "Olá"

nome = "Ana"

turno = "bom dia"
```

**R2.** `tamanho = len(cumprimento) + len(nome) + len(turno)`

**R3.** i, ii, iii, iv
